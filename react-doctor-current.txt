  ⚠ Bugs: Missing effect dependencies ×10

    → Don't blindly add missing dependencies. Read the hook
    callback first.
    
    Bad:
    useEffect(() => {
      setCount(count + 1);
    }, [count]);
    
    Better:
    useEffect(() => {
      setCount((currentCount) => currentCount + 1);
    }, []);

    inside the hook or stabilize it before adding it to deps.
